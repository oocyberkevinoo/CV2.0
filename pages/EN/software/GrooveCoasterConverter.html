GROOVE COASTER CONVERTER (2020)

Application Open Source permettant le décodage de fichiers en format propriétaire, afin d’en modifier le contenu de manière simple.

Il permet aussi la conversion de données d’un format propriétaire à un autre, ainsi que la sortie de fichiers fonctionnels pour l’application propriétaire.


Il est dorénavant l'outil principal utilisé dans le Modding de jeux Groove Coaster afin de modifier la playlist des différents jeux.

Objectifs

- Modification de la playlist du jeu au format binaire.
- Conversions d'une playlist d'une version du jeu à une autre.
- Conversions des données d'une version du jeu à une autre.
