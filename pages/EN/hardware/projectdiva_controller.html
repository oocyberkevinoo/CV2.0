Arcade Controller with Arduino
Project Diva Arcade (2022)

Replica of the original Arcade cabinet from the japanese game "Project Diva Arcade: Future Tone", comaptible with the PC et Nintendo Switch ports.

With the help of the community to reverse engineer the original hardware made for the console ports, we achieved to recreate the original protocol of the 32 sensors on our own hardware, recreating the touch panel of the original cabinet, with the limitations of a console controller's inputs.
This controller also have his own manager's software and a built-in menu with coded LEDs for navigation, so you can sustomise your controller easily.


Repo GitHub


How does the touch panel works?

The touch panel have the same restrictions as a classic controller.

The official SEGA controller
use a 8-bits coded by joystick axes to output the 32 points.
Each bit represent a sensor,
using 4 axes (LX/LY RX/RY) to output 32-bits to the console.

We recreated that function for Arduino:


We use some copper sheet as sensors, connected to MPR121 modules that will result in 32 electrodes.
