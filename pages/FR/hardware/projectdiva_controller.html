Controlleur Arcade sous Arduino
Project Diva Arcade (2022)

réplique de la borne d'Arcade japonaise du jeu "Project Diva Arcade: Future Tone" compatible PC et Nintendo Switch.

Tournant sous Arduino, avec l'aide de la communauté pour faire des tests sur le contrôleur officiel, nous avons pu recréer les fonctions spécifiques qui permettent d'ajouter un pavé tactile 32 points fonctionnels sur un simple code de manette.
Le code de ce pad permet aussi de configurer son contrôleur via des menus tactiles et des codes LEDs, ou un logiciel dédié permettant de modifier à loisir le fonctionnement du controlleur.


Repo GitHub


Comment fonctionne le pavé tactile ?

Le pavé tactile fonctionne avec les restrictions des manettes.

Le pad officiel de chez SEGA recréé
les sorties des joysticks via 8-bits par axes.
Chaque bit représente un capteur,
le tout sur 4 axes (LX/LY RX/RY) afin d'avoir 32-bits disponibles.
